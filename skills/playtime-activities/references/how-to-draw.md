# How to draw (step-by-step)

## What it is

A step-by-step illustrated guide — from a basic shape to a recognizable drawing. The kid copies each step in their own space, building up to the final drawing.

## Age range

**3-12 years.**

| Age | Steps |
|---|---|
| 3-4 yr | 3-4 simple shapes |
| 4-6 yr | 5-7 steps |
| 6-9 yr | 8-12 steps with detail |
| 9-12 yr | 10-15 steps, perspective, manga / character design |

## Skills it builds

- Visual observation
- Fine motor
- Sequencing
- Confidence ("I can draw a ___!")
- Pride in the finished product

## Format spec

A **single-page guide** with:

- Title — "How to Draw a [thing]"
- A **finished example** in the corner so they see the goal
- **Step-by-step grid** — each step shows:
  - The new lines added (highlighted in a different color or bold)
  - All previous lines (lighter / gray)
  - A short instruction ("Add two big circles for eyes")
- **A blank box** at the bottom for the kid to draw their version
- **Variations** — "now try it with: a big mouth, eyelashes, sunglasses, a hat"

## Quality bar

- **Each step is achievable** — no "draw a perfect circle" expectations
- **Visible progress** — each step looks closer to the goal
- **The final result is COOL** — kids show this off
- **Personalized**: draw what THIS kid loves (megalodon, t-rex, princess, robot, knight)
- **Encouragement built in**: "your version doesn't have to look exactly like mine — make it yours"
- **Show the simple shapes** — circles, ovals, triangles — kids learn that drawings start with shapes

## Catalog of high-success topics by age

### 3-5 yr

- A face (circle + features)
- A cat / dog / bunny (simple)
- A fish
- A flower
- The sun
- A house
- A tree
- A simple person (stick + circle head)
- A snail
- A butterfly

### 5-8 yr

- A specific dinosaur (T-rex, brachiosaurus, triceratops)
- A dragon
- A unicorn
- A car / truck / airplane
- A specific animal (horse, dolphin, fox, owl, octopus, shark)
- A robot
- A castle
- A spaceship
- A character with personality

### 8-12 yr

- Manga-style faces / characters
- Anatomy (body proportions)
- Animals with fur / texture
- Vehicles in 3/4 view
- Buildings in perspective
- Hands (hard, but fun)
- Eyes with detail
- Feathers, scales, fur close-up

## Variations

- **Add-a-step** — finish my drawing by adding 3 more details
- **Style pack** — same animal in 3 styles (realistic, cartoon, geometric)
- **Family how-to** — kid + parent each draw the same thing, compare
- **Build-a-creature** — combine random parts to make new creatures

## Material requirements

Printer + pencil + crayons/markers.

## Print considerations

- Steps clearly numbered
- New lines visible (use bold or a tint)
- Generous box at the bottom for kid's drawing
- Color mode: B&W works fine; color is for the kid

## Example brief

> Kid: Maya, 4yr
> Interests: ocean, megalodon
> Theme: ocean week

→ Title: "How to Draw a Megalodon (with Maya)"
→ Finished example top-right corner (small)
→ 5 steps:
  1. Draw a long, fat oval lying down (the body)
  2. Add a triangle on top (the dorsal fin) and one on the side (side fin)
  3. Add a Y-shape at the back (the tail)
  4. Make a curvy line for the mouth — full of triangle teeth!
  5. One small circle for the eye + small dots for nostrils
→ "Now try it!" big box at bottom for Maya's drawing
→ Variations: "Now try: a happy megalodon. A sleepy megalodon. A megalodon eating spaghetti."
